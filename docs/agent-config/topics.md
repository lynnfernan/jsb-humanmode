## How to Add New Topics

When a new topic idea surfaces, add it with:
- **Topic:** The specific angle (not just the theme — "AI and leadership" is a theme, not a topic)
- **Angle:** What's the unexpected or specific take?
- **Notes:** Any context, story seeds, or instructions for the agent
- **Status:** Start at `Backlog`

Move to `Ready` when JSB has approved the topic for generation.
Move to Google Sheet queue when it has a scheduled date.
