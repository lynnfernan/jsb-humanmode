## GIF assembly (EZGif.com)

1. Frame 1 (neutral) — delay: 175 (= 1750ms)
2. Frame 2 (emotional) — delay: 500 (= 5000ms)
3. Loop count: 1 (play once, do not loop)
4. Download and rename per table above
