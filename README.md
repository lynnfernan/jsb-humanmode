## Quick Start

1. Read `docs/brand-strategy.md` for positioning and brand architecture
2. Follow `docs/phase-1-infrastructure.md` day-by-day for the 2-week build
3. Load email sequences from `docs/crm-email-automation.md` into CRM
4. Install analytics per `docs/analytics-setup.md`
5. Brief website designer and PR publicist using `docs/brand-strategy.md` + `docs/content-strategy.md`
